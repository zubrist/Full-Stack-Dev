Prototype-based Inheritance in JavaScript

Introduction

Prototype-based inheritance is a fundamental concept in JavaScript. Unlike classical inheritance found in languages like Java or C++, JavaScript uses prototype-based inheritance, where objects inherit properties and methods directly from other objects.

Example: Creating Objects with Prototypes

In JavaScript, every object has a prototype. You can create an object with a specific prototype using the Object.create() method. Let's create a parent object and then create a child object that inherits from the parent: 


                // Parent object
                const personPrototype = {
                    greet: function() {
                        return 'Hello, ' + this.name + '!';
                    }
                };
                
                // Child object inheriting from parent
                const john = Object.create(personPrototype);
                john.name = 'John';
                
                console.log(john.greet()); // Output: Hello, John!

Understanding Prototype Chain

When you access a property or method on an object, JavaScript first checks if the object itself has that property or method. If not found, it looks up the prototype chain until it finds the property or method or reaches the end of the chain (Object.prototype).

Let's demonstrate this with an example: 


                // Parent object
                const animal = {
                    sound: function() {
                        return 'Animal sound!';
                    }
                };
                
                // Child object inheriting from parent
                const cat = Object.create(animal);
                cat.sound = function() {
                    return 'Meow!';
                };
                
                console.log(cat.sound()); // Output: Meow!
