IIFE

IIFE - Immediately Invoked Function Expression : The code is executed as soon as the script is loaded into memory

Example:

                    
                        (function(){
                            console.log("hello");
                        }) ();

Above code is executed as soon as the script is loaded into memory, and it logs "Hello!" to the browser's console

Benefits - Its help us to create local scope without polluting the global namespace, it also helps in preventing conflicts with other scripts that use similar variable

Example:

                    
                        (function (){

                            var a = 10;
                            var b= 20;
                            console.log(a + b);
                        })();
                        console.log(a); //  output : ReferenceError: a is not defined

although 'var' has function scope but its value is not accessible outside of the IIFE

Another use of IIFE is minification - lets understand with an example

Example:

                    
                        function doMath(){
                            console.log("3^4 = ", Math.pow(3, 4));
                            console.log("root 4 =", Math.sqrt(4));
                            console.log("sin(45) =",Math.sin(45));
                        }
                        doMath()

We can minimize this code using IIFE like below:

Example:

                    
                        (function (l,p,r,s){
                            l("3^4 = "+p(3, 4));
                            l("root 4 =" + r(4));
                            l("sin(45) ="+ s(45));
                        })(console.log, Math.pow, Math.sqrt,Math.sin);

Both the above code will gives the output:

                    3^4 =  81
                    root 4 = 2
                    sin(45) = 0.8509035245341184
